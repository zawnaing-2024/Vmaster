🧪 HOW TO TEST:

METHOD 1: Android Studio (Easiest)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Install Android Studio
   Download: https://developer.android.com/studio

2. Open project
   File → Open → Select: VMasterVPN-Android folder
   Wait for Gradle sync (5 minutes)

3. Update API URL
   Edit: app/src/main/java/com/vmaster/vpn/utils/Constants.kt
   
   For emulator:
   const val API_BASE_URL = "http://10.0.2.2:8000/"
   
   For physical device:
   const val API_BASE_URL = "http://YOUR_MAC_IP:8000/"
   
   Find your Mac IP:
   ifconfig | grep "inet " | grep -v 127.0.0.1

4. Run app
   Click: Run → Run 'app' (▶️ button)
   Select: Device or Emulator
   Wait for app to install and launch

5. Test
   • Login with client username/password
   • See VPN accounts list
   • Click account (shows TODO for now)
   • Pull to refresh
   • Logout

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

METHOD 2: Build APK and Install
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Build APK in Android Studio
   Build → Build Bundle(s) / APK(s) → Build APK(s)
   
2. Find APK
   app/build/outputs/apk/debug/app-debug.apk

3. Install on device
   adb install app-debug.apk
   
   Or transfer to phone and install manually
